# ЁЯРИтАНтмЫЁЯРИтАНтмЫ Alley Cat Disassembly ЁЯРИтАНтмЫЁЯРИтАНтмЫ

### meow meow

An attempt to have Claude Code completely reverse-engineer the entire executable of **Alley Cat** by Bill Williams (1983).

The goal is to document and understand all the sprites, memory layout, and name all functions and labels тАФ turning raw disassembly into readable, well-annotated NASM x86 assembly that rebuilds byte-for-byte identical to the original `cat.exe`.

The code is 100% written by LLM, with a lot of guidance from a human.
